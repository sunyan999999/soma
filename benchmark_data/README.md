# SOMA 基准测试数据集文档

## 概述

本目录包含 SOMA 基准测试所用的标准化数据集。每个数据集均有版本号、结构化标注和可复现生成脚本，供第三方独立验证和复现。

## 数据集清单

### 1. 思维拆解数据集 — `soma/test_dataset.json`

| 属性 | 值 |
|------|-----|
| 文件名 | `soma/test_dataset.json` |
| 版本 | 1.1 |
| 问题数 | 50 |
| 分类 | 跨域推理(10) / 深层归因(10) / 经验借鉴(10) / 战略决策(10) / 创新设计(10) |
| 难度 | 1-3 (v1.1新增) |

**用途**: 评估 `WisdomEngine.decompose()` 的思维拆解覆盖率——即 SOMA 能否将复杂问题拆解为恰当的思维规律组合。

**标注字段**:
- `expected_laws`: 预期应触发的思维规律列表（7条规律中选2-3条）
- `reference_approach`: 人类专家的参考解题思路
- `memory_tags`: 预期召回的相关记忆关键词
- `difficulty`: 题目难度（1=基础 2=中等 3=困难）

**评分方式**:
- `decomposition_coverage`: 实际触发规律 ∩ 预期规律 / 预期规律数量
- `law_recall`: 预期规律被命中的比例
- `memory_precision`: 激活tag与预期tag的Jaccard相似度

**方法论**: 每题由人类标注者依据SOMA的7条思维规律体系构建，每题的 `expected_laws` 基于「该问题最适宜用哪些思维规律来分析」的专家判断。

### 2. 大规模检索数据集 — `benchmark_data/retrieval_1k.json`

| 属性 | 值 |
|------|-----|
| 文件名 | `benchmark_data/retrieval_1k.json` |
| 版本 | 1.0 |
| 段落数 | 1000 |
| 查询数 | 50 |
| 领域数 | 20 |
| 生成方式 | `scripts/generate_retrieval_dataset.py` (seed=42) |

**用途**: 评估各记忆系统在大规模数据下的语义召回率和查询延迟。可同时用于 SOMA 自身基准测试和竞品对比测试。

**数据集结构**:
- `passages`: 1000 条中文段落，每条含 `id`、`domain`（20个领域）、`text`、`keywords`
- `queries`: 50 个查询，每条含 `relevant_passages`（ground truth相关段落ID列表）、`domains`（相关领域）

**领域覆盖**: 人工智能、商业管理、经济学、心理学、系统科学、哲学思维、工程实践、生物医学、历史文化、数学、物理、教育、材料科学、农业食品、能源环境、法律、艺术设计、社会学、地理科学、体育竞技

**评分指标**:
- Recall@5/10: 前5/10个召回结果中相关段落的覆盖率
- MRR (Mean Reciprocal Rank): 第一个相关段落的排名倒数的均值
- P50/P95 查询延迟

**可复现性**: 生成脚本固定随机种子 (seed=42)，同一脚本版本始终产出相同数据集。

## 数据使用许可

本基准测试数据集遵循 Apache 2.0 许可协议。数据集中的段落文本为合成生成内容，不包含真实个人数据。

## 贡献指南

向现有数据集添加新问题或新段落：
1. 遵循已有数据格式和标注规范
2. 更新版本号（语义化版本）
3. 更新本文档的版本记录
4. 确保 JSON 格式有效（`python -m json.tool` 验证）
