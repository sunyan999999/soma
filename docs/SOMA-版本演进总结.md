# SOMA 版本演进总结 — 从 v0.1 到 v1.0

> 一句话：SOMA 从一个"记性好"的知识库，长成了一个能记忆、能推理、能协作、能自我觉察、能持续进化的 AI Agent 认知内核。

---

## 六次跃迁，五线交汇

### v0.7.0 — 记忆不再"只进不出"

之前的 SOMA 像个无限扩容的仓库，什么都往里塞。这一版给了它三种人脑才有的能力：

- **会合并**：相似的记忆自动归纳成一条，不让你看重复内容
- **会遗忘**：不重要的、很久不用的记忆自动归档，像人脑的遗忘曲线
- **会学习**：能批量导入外部文档（Markdown、网页等），快速建立知识基础

**一句话**：从"记账本"升级为"会整理的大脑"。

---

### v0.8.0 — 从"找到"到"理解"

上一版解决了记忆怎么存，这一版解决了记忆怎么用。SOMA 开始真正理解记忆之间的关系：

- **发现因果链**：不只告诉你"发生了什么"，还能回溯"为什么会这样"
- **标记矛盾**：当你的记忆中存在互相打架的观点，它能发现并提醒
- **跨域联想**：用A领域的经验解释B领域的问题（比如用生物学解释企业管理）
- **质量自评**：每次回答后会给自己打分，低了就反思改进

性能方面，查询延迟从1秒降到200毫秒，CPU负载从接近满载降到13%。

**一句话**：从"搜索引擎"升级为"推理引擎"。

---

### v0.9.0 — 从一个人到一群人

之前只有一个 SOMA 在思考。这一版让它能组队：

- **专家分工**：可以注册多个 Agent，各自专攻不同领域（技术、商业、心理...）
- **自动路由**：问题来了自动分发给最懂的那个 Agent 回答
- **共识机制**：多个专家意见不一致时，投票或综合出最优解
- **独立进化**：每个专家在自己的领域里越用越强，同时分享群体经验

**一句话**：从"一个人的智慧"升级为"一群人的协作"。

---

### v0.9.1 — 多了一个"自知之明"

这是最微妙也最特别的一次升级。SOMA 开始察觉用户的思维模式：

- 当你连续5轮只用同一套思维框架分析问题，它会温柔提醒你"要不要换个角度？"
- 不强制、不打断、不改变你的决策——只是在回答末尾附加一行小字提示
- 默认关闭，需要的人自己打开

**一句话**：SOMA 不仅能陪你思考，还能帮你看见"自己是怎么想的"。

---

### v1.0.0 — 记忆有了"层次感"，五线交汇

这是里程碑版本。v1.0.0 的核心突破是**记忆分层**——让记忆从碎片化走向结构化：

- **L1 情节碎片**（原有）：一条一条的具体信息
- **L2 场景块**（新增）：自动把相关碎片聚合成"场景"——比如"用户在做Python数据分析项目"
- **L3 用户画像**（新增）：从多个场景中提炼用户特征——偏好什么语言、擅长什么、缺什么
- **自动捕获管道**：整个过程全自动，你只管正常使用，SOMA 在后台默默聚合提炼

但这不只是"记忆的升级"。v1.0.0 是**五条能力线的交汇点**：

| 能力线 | 核心交付 |
|--------|---------|
| **记忆** | 三层体系（碎片→场景→画像）+ 合并 + 遗忘 + 外部导入 |
| **推理** | 因果链 + 冲突检测 + 跨域类比 + 质量自评 + 图谱扩展 |
| **协作** | 多Agent团队 + 专家路由 + 共识协议 + 分布式进化 |
| **进化** | 反思→调权→固化→分享 + 自动场景捕获 |
| **工程** | 511测试 + 五维基准 + 竞品对比 + 统计输出 |

**所有新功能默认关闭，旧代码无需任何修改就能运行**——这一原则从 v0.7.0 贯彻到 v1.0.0，从未打破。

基准测试：记忆 97.6 / 智慧 87.3 / 进化 60.2 / 伸缩性 100.0 / **综合 85.5**。
