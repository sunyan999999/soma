# 贡献者公约 / Contributor Covenant

## 我们的承诺

为了营造一个开放和欢迎的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、种族、性别认同与表达、经验水平、国籍、个人外貌、宗教、性取向与社会经济地位，参与本项目不会受到骚扰。

## 我们的准则

有助于创造积极环境的行为包括：
- 使用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员展现同理心

不可接受的行为包括：
- 使用性暗示语言或图像，以及令人不适的性关注或挑逗
- 挑衅、侮辱/贬损性评论，人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 在专业场合被合理认定为不当的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受行为采取适当和公平的纠正措施。

## 执行

侮辱、骚扰或其他不可接受的行为可以通过联系项目团队报告至 [插入邮箱]。所有投诉都将被审查和调查，并将被视为必要且适合情况作出回应。
