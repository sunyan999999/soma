## 4. Bug 修复记录

| 问题 | 根因 | 修复 |
|------|------|------|
| `_text_overlap` 对中文始终返回0 | 正则 `[一-鿿]+` 将无空格中文整段匹配为单个token | 改用 jieba 分词 |
| `_extract_json` 列表解包崩溃 | `enumerate` 返回 `(index, item)` 被错误解包为 `(k, item)` | 拆分列表/字典两个分支处理 |
| `forgetting.py` archive 时 `row.get()` 崩 | `sqlite3.Row` 没有 `.get()` 方法 | 改用 `row["key"] if "key" in row.keys() else default` |
