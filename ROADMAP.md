## 进行中

### v0.9.0 — 多智能体协作（预计 2026-07）

- [ ] **记忆隔离与共享**：每个子 agent 有独立记忆分区，同时通过「公共记忆区」共享知识
- [ ] **共识形成协议**：多 agent 对同一问题提出不同观点时，自动协调形成共识
- [ ] **专家分配器**：根据问题类型自动路由给「最懂这个」的记忆分区
- [ ] **分布式权重演化**：子 agent 的权重演化独立进行，定期合并到全局模型

### v1.0.0 — 生产就绪（预计 2026-09）

- [ ] **完整文档站**：API 参考 + 集成指南 + 最佳实践 + 视频教程
- [ ] **官方集成包**：LangChain、LlamaIndex、CrewAI、AutoGen 官方集成
- [ ] **性能基准**：10 万+ 记忆条数下 < 10ms 检索延迟
- [ ] **安全审计**：第三方安全审查 + SOC2 类型自查
- [ ] **企业功能**：RBAC、审计日志、SSO、私有部署指南

## 永远不做

- ❌ 云托管服务（SaaS）—— SOMA 永远保持本地优先
- ❌ 向量数据库替换 —— SOMA 不是数据库，是认知框架
- ❌ 封闭模型绑定 —— SOMA 支持任何 LLM，也支持纯本地 ONNX
